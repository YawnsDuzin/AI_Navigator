# AI Service Hub (AI 서비스 내비게이터)

AI 관련 서비스들을 카테고리별로 체계적으로 관리하고, 쉽고 빠르게 접근할 수 있도록 돕는 웹 애플리케이션입니다. 직관적인 UI와 드래그 앤 드롭 기능을 통해 나만의 AI 도구 모음을 구성할 수 있습니다.

## ✨ 주요 기능

*   **서비스 관리**: 즐겨찾는 AI 서비스의 이름, URL, 아이콘을 등록하고 수정, 삭제할 수 있습니다.
*   **카테고리 관리**: 서비스들을 그룹화할 카테고리를 생성하고 관리합니다. (이모지 아이콘 지원)
*   **드래그 앤 드롭**:
    *   서비스 카드의 순서를 자유롭게 변경할 수 있습니다.
    *   카테고리 간 서비스 이동이 가능합니다.
    *   카테고리 자체의 순서도 변경할 수 있습니다.
*   **강력한 검색**: 등록된 서비스를 실시간으로 검색할 수 있습니다.
*   **테마 지원**: 눈이 편안한 **다크 모드**와 깔끔한 **라이트 모드**를 지원합니다.
*   **반응형 디자인**: PC, 태블릿, 모바일 등 다양한 디바이스 환경에 최적화된 화면을 제공합니다.
*   **데이터 백업**: 설정한 데이터를 JSON 파일로 내보내거나 가져올 수 있어, 데이터를 안전하게 보관하고 다른 기기로 옮길 수 있습니다.
*   **뷰 모드 전환**: 그리드 뷰와 리스트 뷰 중 선호하는 방식을 선택할 수 있습니다.

## 🛠️ 기술 스택

*   **HTML5**: 시맨틱 마크업 구조
*   **CSS3**:
    *   Modern CSS (Flexbox, Grid, CSS Variables)
    *   Glassmorphism 디자인 스타일 적용
    *   반응형 미디어 쿼리
*   **JavaScript (Vanilla)**:
    *   외부 라이브러리 의존성 없는 순수 자바스크립트 구현
    *   LocalStorage를 활용한 데이터 영구 저장

## 🚀 실행 방법

1.  이 저장소를 클론하거나 다운로드합니다.
2.  `index.html` 파일을 웹 브라우저(Chrome, Edge, Safari 등)에서 엽니다.
3.  별도의 서버 설치 없이 바로 사용할 수 있습니다.
